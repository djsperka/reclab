This folder contains the input tables used from July 26, 2017 until (further notice).

The naming convention for these files is the following: 

AM or SP: amplitude modulation task or spatial task

17 or 33: the frequency of S1, in Hz

all or easy: 7 S2 values (all) or 4 S2 values (easy)

71: octave span of the stimuli for AM task

spkr5 or spkr11: the speaker S1 is played from.
